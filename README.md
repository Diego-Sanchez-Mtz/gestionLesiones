# Proyecto Gestión de Lesiones
___

### Descripción:
Este programa pretende gestionar el registro y seguimiento a las lesiones que sufren las jugadoras del equipo representativo de la UNAM 
de fútbol soccer femenil, así como el registro y seguimiento a planes de rehabilitación que incluyen sesiones de terapia.

## Descripción del problema
La gestión de jugadoras lesionadas puede ser un proceso complicado al realizarse de forma manual. Además, el no conocer la disponibilidad 
o el tipo de lesión que tiene la jugadora puede llevar a exponerla a sufrir una lesión más grave que la deje sin jugar mucho más tiempo.  
Un sistema que ayude a llevar un registro automatizado puede ser útil para la toma de decisiones estratégicas para el equipo de futbol. 

## Contexto
En el equipo representativo de futbol femenil de la UNAM la gestión de jugadoras puede resultar complicada debido a las lesiones que presentan 
algunas de las jugadoras.
